# go-crud-sample

サーバサイド講義用のサンプルコード

## Requirement

### Versions

Go v1.15
macOS 64 bit
Windows 10
Docker Engine v19.03``

### Directories

参考: golang-standards/project-layout

```shell
go-crud-sample
├── bin         バイナリ出力
├── build       Docker 等のパッケージファイル
├── cmd         main 関数が定義されたファイル
├── configs     設定ファイル
├── db          ローカル DB
├── deployments デプロイファイル
├── docs        各種ドキュメント
├── internal    app 内部のコード
└── scripts     作業用スクリプト
```
